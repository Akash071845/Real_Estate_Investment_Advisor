Ths steps to execute this Project is as follows
FIRST STEP:- Execute the EDA file, Which will save a new dataset called Cleaned_Data.
SECOND STEP:- Execute the Regression file, Which will save a new dataset called Regression_Data. And the Mlflow will log the models along with parameters and metrics
THIRD STEP:- Execute the Classification file, Which will save a new dataset called Classification_Data
FOURTH STEP:- Before executing the app.py, first you have to copy the run id of the best regression model from mlflow and paste it in the app.py code
FINAL STEP:- Then Execute the file app.py, to see the deployed model
